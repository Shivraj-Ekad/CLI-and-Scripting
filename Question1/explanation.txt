mkdir module_lab

→ Creates module_lab Directory

cd module_lab/

→ Moves into the module_lab working directory.

nano analyze.sh

→ Creates and opens the script file for editing.

edited file then CTRL + O → Enter → CTRL + X

→ Gives execute permission to the script.

→ Creates a sample file to test script.
