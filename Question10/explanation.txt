mkdir Question10

cd ~/Question10

nano signal_demo.c
Creates signal handling C program file.

Adds signal handling logic for SIGTERM and SIGINT.

gcc signal_demo.c -o signal_demo
Compiles signal handling program.

./signal_demo
Runs program and waits for signals from child processes.
